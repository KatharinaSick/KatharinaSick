## What I'm building

### [offon.dev](https://offon.dev)

<img src="https://offon.dev/nyx.webp" align="left" width="80" hspace="15" vspace="5">

A vendor-neutral open source community I've been helping build from the ground up. Every month we release a new challenge series: hands-on, browser-based & no local setup needed. Think broken pipelines, misconfigured Kubernetes clusters, real-world scenarios. When I'm not creating adventures, I'm reviewing them or helping other builders shape theirs.
